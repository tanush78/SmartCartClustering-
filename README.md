# SmartCartClustering-
A data science project that analyzes retail customer data to uncover distinct customer segments using unsupervised machine learning (clustering). 
The dataset includes demographic details (age, education, marital status, income, number of children), purchasing behavior (spending across categories like wine, meat, fish, sweets, and gold products),
and engagement metrics (purchases via web, catalog, and store, deal usage, campaign responses, and complaints) make it small
